# 跨境电商商品数据分析项目
E-commerce Data Analysis Project

## 📌 项目介绍
基于真实跨境电商商品数据，完成全流程数据分析：
数据读取 → 数据清洗 → 多维度分析 → 可视化 → 业务报告

适用于电商品类分析、定价分析、利润分析、库存策略支持。

## 🛠 技术栈
- MySQL 数据存储
- Python 数据分析
- Pandas 数据清洗与统计
- Matplotlib 可视化
- SQLAlchemy 数据库连接

## 📊 数据集
来源：阿里云天池
表名：may-2022
数据量：1330 条

## 🔍 分析内容
- 商品品类结构分析（TOP3 品类）
- 拿货价 / 售价 / 利润空间分析
- 数据清洗（去重、缺失值、异常值、Nill 过滤）
- 可视化图表输出
- 业务运营建议

## 🎯 分析结果
- 核心品类：Kurta、Kurta Set、Tops
- 商品利润空间稳定，盈利性良好
- 品类结构集中，适合重点备货核心品类

## 🚀 运行方式
1. 配置 MySQL 连接信息
2. 运行 shujufenxi.py
3. 自动输出分析结果与图表
